## 👨‍🏫 About

데이터와 교육을 연결하여 더 나은 학교를 설계합니다.  
교육측정, 학습분석(Learning Analytics), 사회네트워크분석(SNA)을 통해 학생의 성장과 교사의 전문성, 학교 조직의 변화를 연구합니다.

*Connecting data and education to design better schools. Specializing in Educational Measurement, Learning Analytics, and Social Network Analysis (SNA) to foster student growth, teacher professionalism, and school improvement.*
